# Customer Shopping Behaviour Analysis

## Overview

This project performs an end-to-end **data analytics workflow** to analyze customer purchasing behavior. The analysis focuses on identifying patterns in customer spending, product popularity, discount usage, and customer segments.

The project uses **Python for data analysis, PostgreSQL for SQL queries, and Power BI for dashboard visualization** to generate insights that help understand customer behavior and purchasing trends.

---

## Dataset

The dataset contains information about customer purchases, including:

* Customer demographics (age group, gender)
* Products purchased
* Purchase amount
* Discounts applied
* Shipping type
* Review ratings
* Previous purchase history
* Subscription status

This data helps analyze **customer segments, spending behavior, and product performance**.

## Project Workflow / Steps

### 1. Data Loading (Python)

* Imported the dataset into Python using **Pandas**.
* Examined dataset structure, data types, and missing values.

### 2. Exploratory Data Analysis (EDA)

* Analyzed customer spending patterns.
* Visualized relationships between variables.
* Explored product categories, customer demographics, and purchasing trends.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Standardized column formats.
* Prepared the dataset for SQL analysis and visualization.

### 4. SQL Analysis (PostgreSQL)

The cleaned dataset was imported into **PostgreSQL**, where multiple analytical queries were performed to extract insights such as:

* Revenue generated by **male vs female customers**
* Products with the **highest average ratings**
* Comparison of **shipping types and purchase amounts**
* Spending patterns of **subscribed vs non-subscribed customers**
* Identification of **top purchased products**
* Customer segmentation into **New, Returning, and Loyal customers** based on previous purchases 

### 5. Power BI Dashboard s
* Revenue by age group

The dashboard enables quick exploration of customer behavior and product performance.

## How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/customer-shopping-analysis.git
```

### 2. Install Required Python Libraries

```bash
pip install pandas numpy matplotlib seaborn psycopg2
```

### 3. Run the Jupyter Notebook

Open the notebook file and run all cells to perform:

* Data loading
* Exploratory Data Analysis
* Data cleaning

### 4. Run SQL Queries

Import the cleaned dataset into **PostgreSQL** and execute the SQL queries provided in the project.

### 5. Open Power BI Dashboard

Open the **.pbix file** in Power BI Desktop to explore the interactive dashboard.
